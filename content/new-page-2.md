# Getting Started

Welcome to the Markdown Site Generator!

## What is this?

This is a simple tool that lets you write markdown content and instantly preview it as a website.

## Features

- **Live Preview** - See your changes in real-time
- **Clean URLs** - Each page gets a beautiful URL like `/site/getting-started`
- **Simple Editor** - Just write markdown, no complex formatting needed
